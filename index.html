Panduan Instalasi

Cara Download & Install Aplikasi Siswa

Ikuti langkah mudah berikut untuk menginstall aplikasi di HP Android kamu.

1

Buka Landing Page

Buka halaman ini di browser HP Android kamu.

2

Download APK

Tap tombol "Download Aplikasi Siswa" dan tunggu file terunduh.

3

Izinkan Instalasi

Buka Pengaturan → Keamanan → aktifkan "Instal dari sumber tidak dikenal".

4

Install & Login

Buka file APK → Install → Buka aplikasi → Login dengan akun Google.

Hanya Android ~25 MB Gratis

Tentang Aplikasi

Aplikasi ini dikembangkan sebagai bagian dari penelitian skripsi di bidang teknologi pendidikan.
